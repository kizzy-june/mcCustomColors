CustomColors
=======
Minecraft mod that allows you to change sky, cloud, foliage, water, grass and fog color!+

Also allows you to use the default minecraft colors while the mod is installed.

That's it, you can configure the mod by editing the mod's config file or via the configuration menu that is included in Neoforge.

Known issue:
The foliage color option does not work with Birch, Spruce, Cherry and Pale Oak trees.
